Právomoci užívateľov

Všetci

  • Prezerať si profily lovcov a mamutov
  • Prezerať si históriu lovov, správ a hliadok
  • Prezerať si lovecké jamy

Hunter

  • Zúčastniť sa lovov a hliadok
  • Ak je na love: podať správu lovu
  • Ak je na hliadke: pridať mamuta, poslať správu o spozorovaných mamotoch

Officer

  • Upraviť históriu lovov a hliadok
  • Upraviť profily lovcov
  • Upraviť alebo vytvoriť profily mamutov
  • Zmeniť rolu užívateľa
  • Pridať lovecké jamy

Manager

  • Určit loveckú skupiniu (spustiť lov)
  • Určit a ukončiť hliadku

Administrátor

  • Všetky právomoci (pomocou rozhrania pre administrátora)

Implementace

Use-case

Use-case je od pôvodného zadania mierne upravený. Pôvodné zadanie bolo pripravené pre jaskynných ľudí a tým pádom so samotnou databázou dokázal narábať iba jeden uživateľ (jediná gramotná osoba). Aktuálny use-case zodpovedá právomociam uživateľov a plne pokrýva pôvodný use-case.

Stručná dokumentace k implementaci, která popisuje, které PHP skripty implementují jednotlivé případy použití uvedené v use-case diagramu z IDS. V případě, že bylo nutno při implementaci učinit změny v návrhu, musí být v dokumentaci zdůvodněny.

Modely

Instalace

Postup spustenia

  • Stiahnuť archív (git clone https://github.com/edynox/iis)
  • Nainštalovať knižnice (npm install)
  • Vytvoriť schému databázy (python3 manage.py makemigrations)
  • Aplikovať schému databázy (python3 manage.py migrate)
  • Vytvorenie administrátora (python3 manage.py createsuperuser)
  • Spustiť server (python3 manage.py runserver)

Softwarové požiadavky

  • Python 3.x
  • Django 1.11.x
  • Javascript package manager (npm)

Známé problémy

  • Riešenie plne iplmentuje zadanie z IDS (upravený use-case diagram)
  • Riešenie plne iplmentuje zadanie z IIS
